# 一款快速更新Apk到安卓应用市场的工具

## 功能特点：

1. 使用应用市场提供的Api传包功能，安全，稳定，快速
2. 代码开源
3. 完全免费
4. 使用kotlin 编写，支持Windows 和Mac OS

1. 使用compose desktop 开发
2. Ant design 设计风格
